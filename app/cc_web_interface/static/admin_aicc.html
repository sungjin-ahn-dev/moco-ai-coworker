1. 상담사 연결 라우팅 즉시 반영

고객을 상담사에게 자동 연결할지 여부와 기준을 설정합니다.
고객을 연결할 상담사 회선 번호. 하이픈 없이 숫자만 입력하세요.
AI가 고객 발화를 N번 연속 못 알아들으면 자동으로 상담사에게 연결합니다. 권장: 3회.

2. 전환 키워드 즉시 반영

고객이 이 표현을 말하면 즉시 상담사로 연결합니다.
고객 발화에 이 키워드가 포함되면 즉시 상담사 연결.
강한 불만이나 욕설 키워드. 감지 시 즉시 상담사 연결됩니다.
AI 응답에 이 표현이 포함되면 인식 실패로 간주하고 카운트합니다. 임계값 도달 시 자동 전환.

3. 안내 멘트 (G01~G51) 다음 통화부터 적용

AICC 시나리오 v1.2 기본흐름 스크립트 15단계. AI가 각 상황에서 정확히 이 멘트로 말합니다.
말투 원칙, 대화 흐름 규칙, TTS 숫자 처리 규칙. 어르신 대상이라 짧고 천천히 읽히는 문장 권장.
전화 인입 즉시 재생.

4. 운영시간 즉시 반영

상담 가능한 요일과 시간을 설정합니다. 시간 외 호출은 자동 안내 후 종료됩니다.
사용요일시작종료

4-1. 점심시간 즉시 반영

운영시간 안에서도 점심시간엔 자동 안내 멘트(상담사 연결 불가)가 나갑니다.
기본값: 13:00 ~ 14:00 (시나리오 v1.2 기준).

5. 야간 자동 휴식 즉시 반영

매일 지정 시간 동안은 운영시간이어도 자동 휴식 안내가 나갑니다.
기본값: 0시 ~ 4시. 시작이 종료보다 늦으면 자정을 넘긴 시간대로 처리됩니다.

6. 휴무일 즉시 반영

특정 날짜를 휴무일로 지정합니다. 해당 날짜에는 휴무일 안내 후 통화가 종료됩니다.

7. 사후 안내 SMS 즉시 반영

통화 종료 후 고객에게 자동 발송하는 안내 문자. 헤더 + AI가 정리한 상담 요약 + 풋터로 합성됩니다.
예: "안녕하세요. 상담 전화 주셔서 감사합니다."
예: "궁금하신 점 생기시면 언제든지 다시 문의 주세요. - 제품A 고객센터 1588-0000"
의미 있는 상담이 없었을 때 헤더와 풋터 사이에 들어가는 문구.
고객·AI 합쳐서 이 턴 미만이면 "상담 내용 없음"으로 간주합니다. 권장: 2.
